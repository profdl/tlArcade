# Face Mask

> Part of the [tlArcade](../../../README.md) prototyping platform — mounted
> at `/demos/face-mask`. No `package.json`/build of its own; run everything
> from the repo root (`npm run dev`, then visit `/demos/face-mask`).

Point a webcam at your face and pin native tldraw shapes to tracked
landmarks — stars for eyes, a square for a nose, an arrow that follows a
feature — so they move with you live.

See [CLAUDE.md](CLAUDE.md) for the architecture.
